Getting and Cleaning Data
Data Science Specialization

This repo contains the source code to implement the project of Getting and Cleaning Data course on Coursera. This course is part of the Data Science
spcialization.

In this project, there are three files:
1- run_analysis.R which is the main file that contains all function implementation for reading and cleaning the datasets
2- CodeBook.rmd written in R Mark Down to illustrate every step used in the run_analysis.R file towards generating the output.
3- ReadMe.txt file (this file)
